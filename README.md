
# Odin Library

A simple web-based library management application built with HTML, CSS, and JavaScript. This project allows users to add books to a personal library, view them in a card-based grid layout, and manage their read status or remove them.

## Features

- Add new books with title, author, and read status.
- Display books in a responsive grid of cards.
- Toggle read status for each book.
- Remove books from the library.
- Form to input book details, hidden by default and shown on button click.

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (ES6+)

## How to Use

1. Open `index.html` in a web browser.
2. Click the "Books" button to reveal the form.
3. Fill in the book details (Title, Author, Book_ID, Status) and submit.
4. Books will appear as cards in the library display.
5. Use the "Status" button to toggle read status and "Remove" to delete a book.

## Project Structure

- `index.html`: Main HTML file.
- `style.css`: Stylesheet for layout and design.
- `script.js`: JavaScript for functionality.
